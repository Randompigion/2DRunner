## 2d Runner
(yes i couldn't think of a better name).
A simple game where you use the spacebar to jump to the beat.

# How to play
Press space to jump, and you can do a smaller double jump in the air if you need to. Most jumps are timed to the beat, so you _might_ want to use a headset to play.
